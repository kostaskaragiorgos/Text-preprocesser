## Keybinds

<ul>
<li> Insert a file Ctrl + O </li>
<li> Close a file Ctrl + F5 </li>
<li> Exit Alt + F4 </li>
<li> Remove stop words Alt + R </li>
<li> Remove punctuation Alt + P </li>
<li> Word counter and distibution Alt + W </li>
<li> Words to lower case Ctrl + W </li>
<li> About Ctrl + I </li>
<li> Help Ctrl + F1 </li>
</ul>
